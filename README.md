# Quick start
On your EC2, type
     
     git clone https://github.com/micklinISgood/TomcatEC2.git
     cd  TomcatEC2
     ./bin/catalina.sh run
     
On your browser, paste the following link and click on Tweets Elapse.

     http://yourDNS:8080/elapse/TweetElapse.html
